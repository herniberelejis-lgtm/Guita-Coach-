Guita Coach

Coach financiero personal con IA — overview técnico para el equipo

v0.1 · Mayo 2026

¿Qué es?

💰

Presupuesto en 3 franjas

Divide el sueldo en Necesidades, Gustos y Ahorro según porcentajes configurables.

🤖

Clasificación automática

Clasifica transacciones por reglas locales y, si la confianza es baja, usa Claude AI.

🔔

Alertas inteligentes

Avisa al 75% y 90% del límite de cada franja, con proyecciones a fin de mes.

📧

Sincronización Gmail

Parsea emails de pago automáticamente via OAuth para capturar transacciones.

💳

Mercado Pago

Importa movimientos del mes vía OAuth con un solo clic desde la app.

📊

Proyecciones

Calcula el ritmo diario de gasto y predice si se superará el límite de la franja.

Flujo de uso

1

Onboarding

El usuario ingresa su nombre, sueldo mensual y el % que asigna a cada franja (ej: 50/30/20).

2

Dashboard

Ve las 3 franjas con barras de progreso, alertas activas y las últimas transacciones del mes.

3

Sincronizar fuentes

Conecta Gmail y/o Mercado Pago. Con un clic se importan y clasifican las transacciones automáticamente.

4

Revisar & corregir

En la pantalla de Transacciones puede editar categorías. Cada corrección se guarda como regla personalizada.

5

Proyecciones

La pantalla de Insights muestra si va a pasarse del límite en alguna franja antes de que termine el mes.

Servicios / Agentes

🧠 Clasificador app/services/classifier.py

Clasifica cada transacción en tres pasos: (1) reglas del usuario con prioridad máxima, (2) reglas globales predefinidas para ~40 comercios argentinos, (3) Claude Sonnet como fallback cuando la confianza es menor a 85%. Responde en JSON con category, subcategory, confidence y reason.

🔔 Motor de alertas app/services/alert_engine.py

Se ejecuta tras cada transacción nueva. Calcula el gasto acumulado por franja, lanza alerta warning al 75% y critical al 90%. Si Claude está disponible, agrega un consejo personalizado citando los gastos reales del usuario. También genera alertas de proyección si el ritmo diario excederá el límite.

📧 Gmail app/services/gmail.py

Implementa el flujo OAuth 2.0 con Google. Busca emails de pago en el mes actual, parsea montos y comercios, y los ingresa como transacciones para clasificar.

💳 Mercado Pago app/services/mercadopago.py

Flujo OAuth con MP. Obtiene los movimientos del mes actual y los normaliza al mismo formato de Transaction que el resto de las fuentes.

🌱 Seed de demo app/services/seed.py

Carga 13 transacciones de ejemplo del mes actual para demostrar la app sin necesidad de credenciales reales. Solo corre si la DB está vacía.

Variables de entorno (.env)

# Claude AI — clasificación inteligente
CLAUDE_API_KEY=sk-ant-...

# Gmail OAuth
GOOGLE_CLIENT_ID=...apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-...

# Mercado Pago OAuth
MP_CLIENT_ID=...
MP_CLIENT_SECRET=...

# Servidor
APP_URL=http://localhost:8000
DEMO_MODE=true  # false en producción

La app funciona en modo demo sin ninguna API key. Solo se necesitan las credenciales para conectar fuentes reales.

Cómo arrancar

Setup en 2 pasos

# 1. Clonar / descomprimir el proyecto e ir a la carpeta
cd guita-coach
# 2. Arrancar todo (instala deps, crea DB, carga demo, levanta server)
python run.py
# 3. Abrir en el browser
http://localhost:8000
